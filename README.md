Generate a project wiki from your codebase

## Install

```bash
npm install -g ai-wiki
```

## Usage

```bash
npx ai-wiki ./src/
npx ai-wiki ./lib/
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
