# ai-wiki

[![npm version](https://img.shields.io/npm/v/ai-wiki.svg)](https://www.npmjs.com/package/ai-wiki)
[![npm downloads](https://img.shields.io/npm/dm/ai-wiki.svg)](https://www.npmjs.com/package/ai-wiki)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)


Generate a project wiki from your codebase

## Install

```bash
npm install -g ai-wiki
```

## Usage

```bash
npx ai-wiki ./src/
npx ai-wiki ./lib/
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
